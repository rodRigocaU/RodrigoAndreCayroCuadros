Graphics Development

OpenGL - Rubik's Cube

Interactive 3D Rubik’s Cube developed for the Computer Graphics course at Universidad Católica San Pablo (Department of Computer Science). The project implements real-time 3D rendering and face rotations with keyboard controls, including camera navigation and animated effects (e.g., seismic/shake animation). It focuses on core graphics concepts such as transformations, orientation handling, and interactive input mapping.

Game Development

Color Bump 3D

Multiplayer 3D game developed in Unity with real-time integration of Python and OpenCV for vision-based control. One player moves the ball using head orientation tracking, while the other controls its speed through hand gestures and audio input, demonstrating multimodal interaction and real-time system integration.

Game Development

Operation Metal Lab

Infiltrate a secretive laboratory and uncover the hidden experiments within. Navigate through tight corridors while avoiding enemies and security cameras, collecting keys to unlock new areas and advance deeper into the facility. Operation Metal Lab is a top-down tactical shooter with a limited field of view, challenging players to plan each move carefully. Eliminate enemies strategically, manage visibility, and progress through increasingly dangerous stages as you uncover the lab’s secrets.
