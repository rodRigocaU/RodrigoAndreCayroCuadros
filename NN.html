CNN Agent for StarCraft (Minimap-based)

I built a Convolutional Neural Network (CNN) to play a StarCraft-like RTS by using minimap frames as input. The model learns to map spatial patterns from the minimap (unit positions, movement, map control) into game actions/commands, enabling real-time decision-making from visual information. This project covers data collection from gameplay, preprocessing minimap images, model training/evaluation, and an inference loop integrated into the game environment.
